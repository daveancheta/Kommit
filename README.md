# 💬 Kommit

Where dev teams commit — together. Real-time chat, video calls, GitHub integration, and task management in one unified workspace. Less tab-switching, more shipping.

---

## 🏗️ Tech Stack

| Layer            | Technology                                                       |
| ---------------- | ---------------------------------------------------------------- |
| **Framework**    | [Next.js 16](https://nextjs.org/) (App Router)                  |
| **Language**     | TypeScript                                                       |
| **Database**     | PostgreSQL via [Supabase](https://supabase.com/)                 |
| **ORM**          | [Drizzle ORM](https://orm.drizzle.team/) + Drizzle Kit           |
| **Auth**         | [Better Auth](https://www.better-auth.com/) (email + GitHub OAuth) |
| **State**        | [Zustand](https://zustand-demo.pmnd.rs/)                        |
| **UI**           | [shadcn/ui](https://ui.shadcn.com/) + [Radix UI](https://www.radix-ui.com/) |
| **Styling**      | [Tailwind CSS v4](https://tailwindcss.com/)                      |
| **Animations**   | [Motion](https://motion.dev/)                                    |
| **Media**        | [Cloudinary](https://cloudinary.com/)                            |
| **Notifications**| [Sileo](https://www.npmjs.com/package/sileo) (toast notifications) |
